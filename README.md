# BugReportery

**Мастерская баг-репортов** — веб-инструмент для создания, сохранения и экспорта баг-репортов. Учебный проект для портфолио.

## Важное ограничение

- Проект является учебным и демонстрационным.
- Регистрация в приложении доступна только для email домена `@qatest.tech`.

## Возможности

- Создание репорта с обязательными полями (ID, заголовок, описание, шаги воспроизведения, ожидаемый/фактический результат)
- Сохранение репортов в `localStorage`
- Просмотр, загрузка и удаление сохранённых репортов
- Экспорт репорта в PDF (с возможностью вложения скриншотов)

## Технологии

- HTML5, CSS3, JavaScript (ES6)
- Библиотеки: html2canvas, jsPDF

## Как запустить локально

1. Клонируйте репозиторий:  
   `git clone https://github.com/ваш_логин/BugReportery.git`
2. Откройте `index.html` в браузере.
